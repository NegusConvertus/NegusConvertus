# Negus-Convertus
All in one file converter. For the people, by the people.
FIle extension convert:
Audio > MP3, WAV, OPUS
Video > MP4, AVI, DV, MKV, MOV
Gambar > JPG, WEBP, PNG
Dokumen > PDF, DOCX
Fitur:
Audio > Audio
Audio > Video
Video > Video
Video > Audio
Gambar > Gambar
Dokumen > Dokumen
